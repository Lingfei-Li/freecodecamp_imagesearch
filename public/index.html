Image Search Microservice

User Stories:

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

Endpoints:

  • Query endpoint: /query/:queryString
    • Search for images with keyword "lolcat" https://lingfei-imagesearch.herokuapp.com/query/lolcat
  • History endpoint: /history
    • Look at all search history: https://lingfei-imagesearch.herokuapp.com/history
    • Delete history with id = <id>: send DELETE request to https://lingfei-imagesearch.herokuapp.com/history/<id>
    • Delete all history send DELETE request to https://lingfei-imagesearch.herokuapp.com/history/
